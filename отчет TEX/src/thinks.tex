\section{Заключение}

В ходе выполнения лабораторной работы был реализован полноценный прототип поисковой системы, включающий:
\begin{itemize}
    \item автоматический сбор тематического корпуса;
    \item лингвистическую обработку текста (токенизация и стемминг);
    \item построение и использование булева индекса;
    \item поддержку булевых запросов с логическими операциями;
    \item анализ корпуса на соответствие закону Ципфа;
    \item механизм быстрого запуска за счёт сохранения состояния.
\end{itemize}
