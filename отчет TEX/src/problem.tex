\section{Цель работы}

Целью данной работы является разработка прототипа поисковой системы, ориентированной на тематический корпус документов. В рамках проекта реализованы следующие ключевые компоненты:
\begin{itemize}
    \item автоматический сбор корпуса статей с заданных веб-сайтов;
    \item предварительная обработка текста (токенизация и стемминг);
    \item построение булева индекса;
    \item реализация булева поиска с поддержкой логических операций;
    \item анализ распределения терминов в корпусе на основе закона Ципфа;
    \item обеспечение автоматического запуска и быстрой загрузки системы.
\end{itemize}
